# Mizu Presence

Mizu Presence is a selfbot of presence discord in discord.js v13

## How to install ?

You need to have Node.JS 16+
``````bash
git clone https://github.com/Xiaotoxdev/MizuPresence.git
cd MizuPresence
npm i
``````

## How to start ?
```bash
node app.js # To start the presence
```
